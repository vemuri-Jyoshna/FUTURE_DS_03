# 📊 Marketing Funnel & Conversion Performance Dashboard

## 📌 Project Overview

This project analyzes marketing funnel data to identify conversion drop-offs, channel performance, and opportunities to improve lead-to-customer conversion.
The dashboard helps businesses understand how visitors move through different stages of the marketing funnel.

## 🎯 Objectives

* Analyze marketing funnel performance
* Identify conversion drop-offs
* Evaluate marketing channel effectiveness
* Improve lead-to-customer conversion rates

## 🛠️ Tools Used

* Power BI
* Excel
* Data Visualization
* Data Analysis

## 📊 Key Metrics

* Total Visitors: 502K
* Total Leads: 73K
* Total Opportunities: 28K
* Total Customers: 14K
* Overall Conversion Rate: 3%

## 🔍 Key Insights

* Google Ads generated the highest number of leads.
* Social Media produced significant visitor traffic.
* Conversion rate drops after the lead stage.
* Winter Promo campaign generated the highest customers.
